Basic

git init - ініціалізувати пустий репозиторій

git status - стан репозиторію

git add - додає вміст робочого каталогу в індекс

git commit - комітить всі зміни і зберігає їх

git log - показує історію комітів

git checkout - переключення гілок і вивантаження їх вмісту в робочий каталог

git branch - менеджер гілок (перечесляє, створює нові, видаляє та переіменовує)

git push - відправляє локальну гілку на віддалений репозиторій

git pull - завантажує всі зміни з віддаленого репозиторію і об'єднує їх з локальним

git merge- зливає одну або декілька гілок в локальну

git clone - клонувати репозиторій з github

.gitignore файл

.gitignore - файл для налаштування ігноровання файлів GitHub для одного репозиторію

Advanced

git revert - операція для безпечної відміни змін

git reset - для відміни локальних змін у репозиторії Git

git rebase - спосіб перенести зміни з однієї гілки в іншу

git cherry-pick - вибірково призначити коміти Git до поточної робочої гілки HEAD

git stash - дозволяє на час «відложити в архів» (або відкласти) зміни, зроблені в робочій копії
